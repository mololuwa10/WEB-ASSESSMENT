Software Engineering

Software engineering is an engineering method to systematics application software development. A software engineer is a person who designs, develops, maintains, tests, and evaluates computer software using software engineering concepts.

Programming

The process of doing a certain computation, usually by designing/building an executable computer programme, is known as computer programming. Programming entails duties including analysis, algorithm generation, profiling algorithm accuracy and resource usage, and algorithm implementation.
